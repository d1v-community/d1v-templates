# d1v-templates

Template workspace for D1V-managed starters.

## Available templates

- `remix-neon-auth-pay`: Remix starter with passwordless email auth, Neon/Postgres, `/pricing`, and hosted checkout integration.
- `remix-neon-auth`: Remix starter focused on passwordless auth and Neon/Postgres without the payment layer.
- `sui-nextjs-auth-template`: Next.js starter for Sui ecosystem projects with wallet auth.
- `html-template`: Minimal static HTML starter for quick pages and prototypes.

The canonical template registry is [`templates.json`](/Users/apple/project/d1v_sever/d1v-templates/templates.json).
